Unity Slider Pack (Gothic Horror Edition)
---------------------------------------------
Assets:
- slider_track.png (background)
- slider_fill.png (foreground fill)
- slider_handle.png (knob)

Style: brass-metallic tone, dark gothic ambience, perfect for psychological horror menus.

Unity Setup:
1. Import textures as Sprite (2D and UI)
2. Set Filter Mode: Bilinear, Compression: None
3. Recommended slider structure:
   - Background -> slider_track
   - Fill Area -> Fill -> slider_fill
   - Handle Slide Area -> Handle -> slider_handle
4. Use Text color near #DCCAA0 for consistency.
